# gomidata

このリポジトリでは、日本国内の各自治体が公開している家庭ごみ・資源の分別例を、  
アプリケーションから容易に利用できるようにすることを目的として、

* 共通データフォーマット仕様の定義
* 分別データ取得プログラムの提供

を行います。

### データフォーマット仕様

[gomidata-spec](gomidata-spec.md)

### データ取得プログラム

対象自治体名 | プログラムファイル名
-------------|------------------------
豊川市       | [gomiget_toyokawa.py](gomiget_toyokawa.py)
